# Power BI Healthcare Analytics Dashboard

An interactive Power BI dashboard analyzing hospital patient flow, bed occupancy,readmission rates,and department profitability using synthetic healthcare data with a star schema model and DAX-driven clinical and financial KPIs.

---

## 📊 Overview

This dashboard provides a 360° view of hospital operations,clinical outcomes,and financial performance-helping healthcare administrators track capacity, quality of care,and cost efficiency across departments.

The project simulates a two-year period (2024–2025) across 10 hospital departments,45 doctors,3000 patients,and 9000 admission records,modeled in a standard **star schema** for optimal performance and clean DAX.

---

## 🛠️ Tools Used

- **Power BI Desktop** — data modeling, report design & publishing
- **DAX** — KPI measures (readmission rate, occupancy rate, profit margin, etc.)
- **Power Query (M)** — CSV import, data type transformation, star schema shaping
- **Python (pandas, numpy)** — synthetic healthcare dataset generation (star schema: 5 dimension tables + 2 fact tables)
